## Features
- **AI-Generated Art**: Generates unique abstract or pixelated art using lightweight algorithms.
- **Minimalist Design**: Ultra-simplified UI for a distraction-free art experience.
- **Quick and Lightweight**: The app is designed with minimal file size, ensuring fast load times and low data usage.
- **Customizable Art**: Allows users to generate different styles and variations of pixelated art by adjusting parameters.
- **Mobile-Friendly**: Optimized for both desktop and mobile experiences.

##  How It Works
Pixelo uses a small and efficient algorithm that generates pixel art based on predefined or randomized patterns. The algorithm processes simple shapes and colors to create pixelated visuals that can be customized by the user in terms of color, scale, and randomness.

You can tweak various parameters in the `config.js` file, such as:
- **Art Style**: Abstract, grid-based, or random pixel patterns
- **Size**: Adjust pixel size and resolution
- **Colors**: Set a palette or randomize color choices
